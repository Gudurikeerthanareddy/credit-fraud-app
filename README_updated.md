# 💳 Credit Card Fraud Detection App

This project is a Streamlit web application that predicts whether a
credit card transaction is fraudulent or legitimate using a trained
machine learning model.

## 🚀 Features

-   User-friendly interface built with Streamlit\
-   Real-time fraud prediction\
-   Displays fraud probability (if supported by model)\
-   Uses pre-trained model and scaler

## 📂 Dataset

The dataset used for training the model is too large to upload to
GitHub.

Download the dataset (CSV file) from Google Drive:\
https://drive.google.com/file/d/1-r1lNCtH8Fy8wI76MN-varyBPZ_0-Fjn/view?usp=drive_link

After downloading, place the CSV file in your project directory.
